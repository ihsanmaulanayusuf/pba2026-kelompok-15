# Sentiment Analysis on Indonesian Political YouTube Comments

## 📌 Description
This project compares Machine Learning (PyCaret AutoML) and Deep Learning approaches for sentiment classification on Indonesian political YouTube comments.

## 📊 Dataset
Dataset: Indonesian Political YouTube Comments  
Task: Sentiment Classification (Positive, Negative, Neutral)

## 🚀 Models
- Machine Learning: PyCaret AutoML
- Deep Learning: PyTorch (LSTM / IndoBERT)
